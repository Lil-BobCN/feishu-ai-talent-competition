# 圣农 Aily Agent 单节点滚动叙事设计规格

## 1. 范围与边界

本次只优化 `04 / Aily Agent` 节点，不复现整套 01-08 架构流程。

叙事起点是：

```text
03 / 企业数据与决策底座
→ 将结构化信号事件包交给 Aily Agent
```

叙事终点是：

```text
Aily 形成决策就绪事件包
→ 交给 05 / 人工决策与方案审批
```

页面不展示从信息采集、数据传输到 Aily 的完整重演；顶部保留当前 04 节点定位和返回完整方案入口，但正文只讲 Aily 收到事件后的工作过程。

## 2. 权威内容源

业务流程以 `docs/圣农经营智能中枢_完整方案母稿.md` 为准，重点遵循：

- M05 Aily 事件受理与调查循环；
- `观察 → 评估 → 计划 → MCP/人工并行取证 → 校验 → 决定`；
- 决定只能是停止、扩大、等待补证或转人工；
- 局部、区域、跨域、战略是调查范围，不是脚本预判的严重度；
- 等待人工补证时 Agent Run 结束，补证回调按 `event_id` 恢复；
- M06 决策就绪事件包和人工审批边界。

## 3. 页面结构

页面由一个英雄场景和八个滚动叙事场景组成。每一场景占据一个主要视口，内部只表达一个动作；滚动不被锁死，用户可以返回上一场景。

### 3.1 英雄场景：交接发生

展示当前节点标题和最终产物：

```text
03 决策底座输出
→ event_id + 初始异常信号 + 语义版本 + 权限范围
→ 04 Aily Agent
```

结构化信号事件包从左侧进入 Aily，不表现为零散字段漫天飞入。

### 3.2 场景 01：统一事件受理

Aily 读取事件包，拆解问题、背景、已知、未知、原因假设、当前调查范围、工具权限和停止条件。画面明确 Agent 不在此处定性、定责或拍板。

### 3.3 场景 02：观察与评估

围绕价格案例展示互斥原因假设：合法券、规格不可比、价签/POS 不一致、活动审批未同步、未授权低价。每个假设显示待验证证据和当前状态。

### 3.4 场景 03：最小查询计划

Agent 只选下一项最小、必要、已授权查询，优先读取 POS、SAP、CRM 和飞书现场证据。查询计划是可检查中间态，必须显示查询原因和停止条件。

### 3.5 场景 04：局部证据汇入

证据逐条回到事件包，每条证据带来源、时间、质量、权限和对假设的支持关系。局部能解释时，循环可以在此停止。

### 3.6 场景 05：并行补证

当存在缺失字段时，画面分成两条路径：MCP 查询已有系统；飞书向对应责任人发送补证请求。请求显示字段、格式、截止时间、已读/已回复/逾期和预计完成时间。等待时 Agent Run 结束，不持续旋转等待。

### 3.7 场景 06：动态扩大调查范围

只有新证据支持时才从单店扩展到区域、跨域或战略。用范围从点到线再到面的动效表达，但保留分支：证据足够则停止，证据不足且可继续则扩大，权限阻塞则转人工。

### 3.8 场景 07：校验与决定

显示来源、时间、口径、冲突和数据质量校验。循环出口为：停止、扩大、等待补证、转人工。不得把“所有信息完整”作为决策就绪的前提。

### 3.9 场景 08：决策就绪事件包

以结构化对象展示事实、信号、推断、冲突、缺口、证据请求、业务影响、约束、候选方案、停止条件和 `approval_required: true`，并把事件包交给 05 人工决策。

## 4. 动效与交互

- 使用 CSS/GSAP 风格的显式时间线；不把 Remotion/HyperFrames 视频播放器嵌入生产页面。
- 场景进入时，事件包、假设、查询、证据和状态按因果顺序出现；不使用装饰性连续漂浮。
- 进度指示只反映 Aily 内部 8 个场景，不改变主站 Hash；主路由仍为 `#node/aily`。
- 场景切换支持滚轮、触控和键盘；不强制滚动、不捕获滚轮、不阻止返回。
- 当前场景、事件状态、查询范围和下一步输出始终可见；动画完成后状态仍保留。
- `prefers-reduced-motion: reduce` 时关闭路径绘制、错峰进入和自动播放，保留静态顺序与状态。

## 5. 失败与边界

- 缺失数据显示 `unknown`，不补零、不伪造确定性；
- 权限阻塞显示转人工；
- 补证等待显示暂停状态并保留 `event_id`，不让 Agent Run 跨天挂起；
- 证据足够时停止，不为了展示动画强行进入战略调查；
- 任何方案都显示 `approval_required: true`，Aily 不自动批准价格、存栏、产能和市场进入；
- 副本文件和服务端口独立于当前 8124 页面，其他 Agent 的页面不被修改。

## 6. 验收标准

- 副本通过新端口访问，8124 页面保持可用；
- `#node/aily` 进入 Aily 叙事，不出现整套 01-08 正文重演；
- 从 03 交接包开始，八个场景按母稿 M05/M06 展示；
- 能看到停止、扩大、等待补证、转人工四种出口；
- 补证请求和 Agent Run 暂停状态可见；
- 决策就绪包显示 `approval_required: true` 并指向人工决策；
- 桌面 1440/1024/768 和移动 390/320 无页面横向溢出；
- 控制台无 error；
- 减弱动效模式下仍能完整阅读全部状态和内容。
