## 目录说明

| 路径 | 内容 | 用途 |
| --- | --- | --- |
| `research/` | 赛事规则、108 项挑战适配、候选方向、评分和集成研究 | 说明选题依据与方案来源 |
| `docs/` | 方案母稿、设计记录、规格与实施计划 | 方案的可审查正文与决策过程 |
| `output/pdf/` | 已导出的方案 PDF | 用于提交、评审和阅读 |
| `index.html`、`shengnong-nodes/` | GitHub Pages 网站与静态资源 | 在线演示源文件 |
| `tests/` | 网页与文档生成测试 | 验证关键交互和生成流程 |
| `.docx_qa_*/`、`.pdf_qa_*/` | 文档渲染检查截图 | 本地 QA 证据，不作为正式交付 |
| `tmp/` | 渲染中间件、浏览器临时数据、版本试验产物 | 本地临时文件，不应提交 |
| `.playwright-cli/`、`.superpowers/` | 工具运行快照与会话状态 | 本地过程文件，不应提交 |

## 交付边界

本仓库中可作为正式交付候选的内容：

- `research/` 中经整理的竞赛与选题研究
- `docs/` 中的方案正文与评委材料
- `output/pdf/` 中的正式 PDF
- 经确认后保留的测试与验证证据

以下内容默认不进入正式 Git 历史：

- `tmp/`
- 浏览器 profile、缓存、Python 缓存
- 未筛选的 QA 截图和工具快照
- 原始调研交接包和个人报名材料
- 未完成的本地网页原型与浏览器过程文件
